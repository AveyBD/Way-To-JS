## Values and Types

JavaScript is dynamic typed. This means that variables can hold values of any type, without any type enforcement. The following build-in types are available:
1.     string
2.     number
3.     boolean
4.     null and undefined
5.     object
6.     symbol (new to ES6)

## Objects

```javascript
var obj = {
 	a: "js",
 	b: 2
}
obj.a;             // "js"
obj["b"];          // 2
```

## Array
```javascript
var arr = ["js", 2, true];arr[0];            // "js" 
arr[1];            //  2
arr[2];            //  true
```
